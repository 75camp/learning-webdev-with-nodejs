# 使用 Node.js 进行 Web 服务端开发

## 课程内容简介

- 简单 Web 架构与基于 TCP 的 HTTP 协议
- HTTP 协议的数据格式、Header
- Verbs 与状态码
- MIME、内容协商
- HTTP 缓存策略
- Web 服务器通用处理流程与拦截器
- 请求与响应：内容处理
- 基于拦截器的通用 Web 框架 —— KOA
- 基于 KOA 的 MVC 框架 —— ThinkJS
- 工业级 Web 架构
